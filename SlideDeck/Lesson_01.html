Review Questions

Discuss some of the capabilities that a developer can use together with an ArcGIS Developers subscription

Capabilities include the following: • Application registration (taking advantage of OAuth) • Creating hosted services • Monitoring application and credit usage • Downloading SDKs • Testing credit-based services such as Elevation, GeoEnrichment, and Spatial Analysis

What are some of the resources you can access from the API's home page?

Tutorials, conceptual help, the API Reference, samples, community resources, Web application templates, and others.

Discuss scenarios for using the ArcGIS API for JavaScript:

Standard, Compact (Hosted, secure, Local), and custom build using the Web Optimizer.
